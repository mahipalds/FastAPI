# Consultant Management API

**Author:** Mahipal Singh

<p>This API is designed for <strong>HR and staffing professionals</strong> to efficiently manage consultant data. It provides detailed information on consultants, including:</p>

<ul>
  <li>Name</li>
  <li>Location</li>
  <li>Gender</li>
  <li>Pay Rate</li>
  <li>Billing Rate</li>
  <li>Margin</li>
  <li>Expenses</li>
  <li>Profit</li>
  <li>Skills</li>
</ul>

<p>Use this system to:</p>
<ul>
  <li>Streamline recruitment processes</li>
  <li>Optimize staffing and resource allocation</li>
  <li>Track consultant profitability</li>
</ul>

<h2>Features</h2>
<ul>
  <li>Fetch consultant details by ID</li>
  <li>View all consultants</li>
  <li>Sort consultants based on pay rate, billing rate, or margin</li>
  <li>Dynamic path and query parameter support</li>
  <li>Proper error handling using <code>HTTPException</code></li>
</ul>

<h2>Technology Stack</h2>
<ul>
  <li>Python 3.13</li>
  <li>FastAPI</li>
  <li>JSON for data storage</li>
</ul>
