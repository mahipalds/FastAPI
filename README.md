# Consultant Management API
**Author:** Mahipal Singh

<p>This API is designed for <strong>HR and staffing professionals</strong> to efficiently manage consultant data. It provides detailed information on consultants, including:</p> <ul> <li>Name</li> <li>Location</li> <li>Gender</li> <li>Pay Rate</li> <li>Billing Rate</li> <li>Margin (computed)</li> <li>Expenses (computed)</li> <li>Profit (computed)</li> </ul> <p>Use this system to:</p> <ul> <li>Streamline recruitment processes</li> <li>Optimize staffing and resource allocation</li> <li>Track consultant profitability</li> </ul>
<h2>Features</h2> <ul> <li>Create new consultant records</li> <li>Fetch consultant details by ID</li> <li>View all consultants</li> <li>Update existing consultant information</li> <li>Delete consultant records</li> <li>Sort consultants based on pay rate, billing rate, or margin</li> <li>Dynamic path and query parameter support</li> <li>Computed fields using Pydantic (<code>margin</code>, <code>expense</code>, <code>profit</code>)</li> <li>Proper validation and error handling using <code>HTTPException</code></li> </ul>
<h2>Computed Fields Logic</h2> <ul> <li><strong>Margin:</strong> Billing Rate − Pay Rate</li> <li><strong>Expense:</strong> 10% of Pay Rate</li> <li><strong>Profit:</strong> Billing Rate − (Pay Rate + Expense)</li> </ul> <p>These fields are automatically calculated using <code>@computed_field</code> and are not required in API requests.</p>
<h2>API Endpoints</h2> <h3>Home</h3> <ul> <li><strong>GET /</strong> – Welcome message</li> </ul> <h3>About</h3> <ul> <li><strong>GET /about</strong> – API metadata and description</li> </ul> <h3>View All Consultants</h3> <ul> <li><strong>GET /view</strong> – Returns all consultant records</li> </ul> <h3>View Consultant by ID</h3> <ul> <li><strong>GET /consultant/{consultant_id}</strong></li> </ul> <h3>Sort Consultants</h3> <ul> <li><strong>GET /sort</strong></li> <li>Query Parameters: <ul> <li><code>sort_by</code> (pay_rate | billing_rate | margin)</li> <li><code>order</code> (asc | desc)</li> </ul> </li> </ul> <h3>Create Consultant</h3> <ul> <li><strong>POST /create</strong></li> </ul> <h3>Update Consultant</h3> <ul> <li><strong>PUT /edit/{consultant_id}</strong></li> </ul> <h3>Delete Consultant</h3> <ul> <li><strong>DELETE /delete/{consultant_id}</strong></li> </ul>
<h2>Sample Request (Create Consultant)</h2> <pre> { "id": "C01", "name": "John", "city": "Dallas", "gender": "Male", "pay_rate": 50, "billing_rate": 80 } </pre>
<h2>Sample Response</h2> <pre> { "name": "John", "city": "Dallas", "gender": "Male", "pay_rate": 50, "billing_rate": 80, "margin": 30, "expense": 5, "profit": 25 } </pre>
<h2>Data Storage</h2> <ul> <li>Data is stored in a local <code>consultant.json</code> file</li> <li>All changes are persisted automatically</li> </ul>
<h2>How to Run the Project</h2> <ul> <li>Create a virtual environment</li> <li>Install dependencies</li> <li>Run the FastAPI server</li> </ul> <pre> python -m venv myenv myenv\Scripts\activate pip install -r requirements.txt uvicorn main:app --reload </pre>
